# Análise de Score de Crédito de Clientes
# Stack usada 
Python, Pandas, Numpy, Matplotlib, Jupyter Notebook, Scikit-learn.
# Descrição breve 
 Projeto voltado à predição do score de crédito (categórico e contínuo), com foco na
identificação de risco de inadimplência e apoio à tomada de decisões financeiras.
# Objetivo do projeto 
Desenvolver modelos preditivos para estimar o score de crédito de clientes, inter-
pretando os fatores mais relevantes por meio de técnicas explicativas.
# Resultado técnico 
Aplicação de algoritmos de Random Forest e XGBoost. Realizadas 
análises exploratórias (EDA), visualizações, construção de modelos e interpretação dos riscos. Projeto em 
fase de ajustes e validação.
